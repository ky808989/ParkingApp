##Installation
Please install node v6.10.0 and mongo db version v3.4.6
Check command
```
node -v
mongod -v
```
###Dependencies Packages
####Server and API
1. express
2. mongoose
3. body-parser
```
npm install --save express mongoose body-parser
```
####Testing tool
1. mocha
2. nodemon (need to set up)
3. supertest
```
npm install --save mocha nodemon supertest
```
